The R scripts in this folder generate the key figures from the main manuscript and SI:

- **`evolutionary_trajectory.R`**: Produces figures showing the evolutionary dynamics of population composition and overall cooperation rates (e.g., Figures 2 and 5 in the main text)

- **`equilibrium.R`**: Generates figures summarizing the mean cooperation rate (with SD in parentheses) and the dominant strategy at the 30000th tick (i.e., equilibrium), 
across a wide range of BC Ratios and initial liquidity levels (e.g., Figure 4 in the main manuscript).

- **`Figure3.R`**: Reproduces Figure 3 of the main text, which shows the relationship between cooperation and initial liquidity for a specific case where BC Ratio = 2.

All scripts operate on `.csv` files generated by either the C++ or the NetLogo implementation of the model. The main difference lies in how the simulation output is structured:

- The **C++ version** produces one `.csv` file per parameter combination within each experimental batch. Each file contains the full time evolution of the model (up to 30000 ticks,
saved every 250 ticks) for 100 independent runs.
- The **NetLogo version** outputs a single consolidated `.csv` file that includes all simulations in one dataset.

For performance reasons, we recommend using the C++ implementation, which is considerably faster. Accordingly, the R scripts are preconfigured to read the C++ output 
(available in the `Simulation Data` folder). However, once the `all_data` object is loaded, the subsequent analysis is identical regardless of whether the data comes from C++ or NetLogo.
