# Replication Package

This repository includes all materials necessary to replicate the results presented in the main manuscript and supplementary information. It contains:

- **`money.cpp`**: The core C++ implementation of the model. Once compiled, this script enables fast execution of experiments across multiple parameter combinations.
- **NetLogo backup implementation**: A GUI-based version of the model in NetLogo, useful for visualizing key dynamics. This version can also run batch experiments via
BehaviorSpace, mirroring the C++ functionality. Since it is significantly slower than the C++ implementation, we recommend using NetLogo primarily for tinkering with the model or for illustrative purposes.
- **R scripts**: Tools for analyzing the `.csv` files generated by the simulations. These scripts are tailored for the C++ output format but can be easily adapted to process NetLogo output.
- **Simulation data**: Organized into subfolders corresponding to each figure in the main text and supplementary information (SI).

Before running the R scripts, make sure to extract (decompress) the simulation data.
