## Accessing the Output Data

You can download the zipped folder containing all the `.csv` files generated by the C++ simulations used to produce the figures in the main manuscript and Supplementary Information (SI).

- Inside the `Main` subfolder, you will find the `.csv` files corresponding to **Figures 2, 3, and 4** of the main manuscript.  
- Inside the `SI-1` and `SI-2` subfolders, you will find the data used to generate **all figures in the SI file**.

To download a folder, click on the **“Raw”**.
